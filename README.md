# Tripleten web_project_homeland

Projeto desenvolvido no quinto sprint do programa de desenvolvimento web da TripleTen.

## Descrição do Projeto

Landing page responsiva para uma galeria de arte virtual. O foco principal é a aplicação de design adaptativo para diferentes tamanhos de tela (320px, 768px e 1280px).

## Tecnologias Utilizadas

- HTML semântico
- CSS
- Metodologia BEM
- Flexbox
- Media Queries
- Fontes locais (Inter)

## Novos Aprendizados

- Importação e uso de fontes locais
- Design responsivo com media queries
- Breakpoints e adaptação de layouts
